Titres

Les titres des pages ne sont pas affichées par le thème ce qui permet de les nommer comme on le souhaite. Pour en ajouter un il faut ajouter un titre H2 (header en anglais) et ajouter la classe CSS praxion-titre sur la barre latérale.

Ajouter un secteur

Un secteur s'ajoute en l'intégrant dans une structure bien précise. Un secteur se met préférentiellement dans une structure de 3 colonnes pour préserver la mise en page.
Les éléments formant le secteur (ie titre, image, légende ...) doivent être groupés.

L'apparence est donnée en ajoutant la classe mon-secteur au groupe.

Zones de texte

Les textes ont un léger fond bleu dégradé. Pour cela il faut grouper tous les paragraphes appartenant à la même zone.

La classe text-group permet d'ajouter le fond.

Vidéo Youtube

On peut facilement ajouter une vidéo de Youtube à partir de l'interface de Wordpress. Mais pour ajouter une vidéo de Youtube sans prendre plus de place qu'un boutton, il faut ajouter un bloc html.

Il suffit ensuite de nommer le bouton et de remplacer le lien de la vidéo.
Le code à copier se trouve en bas de la section.

Short links

Pour modifier la destination d'un lien, il faut aller dans Outils puis Redirection.

Ensuite appuyer sur modifier.

Si il n'y a pas assez de shortlinks disponibles, il faut créer une nouvelle page

Ensuite sur l'onglet Redirection, ajouter une redirection de la nouvvelle page créé vers la page voulue

Historique des recherches

Pour télécharger l'historique des recherches, il faut se rendre dans les réglages de Relevanssi accessibles à partir de Réglages

Ensuite appuyer sur Journalisation puis exporter le journal au format CSV.

Réglages du chat tawk.to

Les réglages se font sur l'interface https://dashboard.tawk.to/ . Les identifiants et mots de passe ont été envoyés à ym@... (chercher Tawk.to)

Landing page et devis

Url de la landing page : https://praxion.com/bienvenue/ . Elle se configure sur Agile CRM

Le code du devis est donnée par Agile CRM et est collé sur la page d'édition correspondant à la page de devis.

Forcer l'indexation d'un nouvel emploi

Pour forcer l'indexation d'une page, il faut se rendre sur google search console

Ensuite coller l'url de la page à index.

Puis cliquer sur demander/forcer une indexation.
